# 个人作品集网站
## 网站功能
### 主页面
- 个人介绍（姓名、头像、简介、联系方式）
- 技能展示
- 项目展示

### 项目详情页
- 项目名称
- 项目描述
- 技术栈
- 项目链接（GitHub、Live Demo等）

### 博客
- 博客文章列表
- 文章详情页（标题、日期、内容）

### 联系页
- 联系表单（姓名、邮箱、消息）
- 社交媒体链接（GitHub、LinkedIn、Twitter等）

同时，界面需要简洁、美观、响应式，能够在不同设备上正常显示。

## 技术栈
- 前端：Vue+TypeScript
- 样式：Tailwind CSS
- 部署：Vercel
